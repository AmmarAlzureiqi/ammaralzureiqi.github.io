# alzureiqi.dev

Personal portfolio site. Built with Next.js, Tailwind CSS, and deployed to GitHub Pages.

**[alzureiqi.dev](https://alzureiqi.dev)**

## Stack

- Next.js 14 (static export)
- Tailwind CSS
- d3-force (tech stack graph)
- cmdk (command palette)
- Framer Motion
